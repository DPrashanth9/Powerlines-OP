# Power Grid Visualizer - Frontend

React + TypeScript + Vite frontend application using Atomic Design principles.

## Structure

- `src/components/atoms/` - Smallest reusable UI components
- `src/components/molecules/` - Component combinations
- `src/components/organisms/` - Complex UI sections
- `src/hooks/` - Custom React hooks
- `src/services/` - API communication
- `src/types/` - TypeScript type definitions

## Setup

```bash
npm install
npm run dev
```

## Environment Variables

Create a `.env` file:
```
VITE_MAPBOX_TOKEN=your_mapbox_token_here
VITE_API_URL=http://localhost:8000
```
